# ADR 0003: Orivanta Product Name

- **Status:** Accepted
- **Date:** 2026-06-21

## Context

The project began under the temporary working title “Project X.” It now needs
a durable identity before the first repository baseline, public previews, and
product-facing design work.

The product is centered on moving around the Earth, changing perspective, and
exploring geographic information at multiple scales.

## Decision

The product is named **Orivanta**.

The name is a coined combination inspired by **orbit** and **vantage**. Its
working tagline is:

> See the world in motion.

The package name, application interface, page metadata, schemas, generated
dataset descriptions, and project documentation use the Orivanta name.

## Consequences

- Orivanta replaces “Project X” in product and engineering materials.
- The name should be used consistently in future repositories, previews,
  deployment environments, and public communication.
- A formal trademark and domain review remains necessary before commercial
  launch.
- The visual identity remains intentionally minimal until the first audience
  and complete product experience are better defined.
